Do not connect

Library: Wiring
Introduced:  3.2 1 in wiring library
Appearance:

Behavior

Constantly generates a floating or high impedance state.

Pins

A Do Not Connect component has only one output pin whose bit width corresponds to the "Data Bits" attribute

Attributes

This component has only one attribute "Data Bits" seen before.

Poke Tool Behavior

None.

Text Tool Behavior

None.
